# ResolveHub – Complaint Management System

ResolveHub is a full-stack complaint management platform where users can submit, track, and manage complaints through a modern dashboard.

## 🚀 Features

- User login system
- Submit complaints
- Track complaint status
- Complaint dashboard
- View complaint details
- Responsive UI
- REST API backend

## 🛠 Tech Stack

Frontend
- React
- Tailwind CSS
- Axios

Backend
- Spring Boot
- Spring Data JPA
- REST APIs

Database
- MySQL
